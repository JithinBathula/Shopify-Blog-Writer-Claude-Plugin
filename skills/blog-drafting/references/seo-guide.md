# SEO Guide for Shopify Blog Posts

Detailed reference for optimizing blog content for search visibility on Shopify stores.

## Keyword Research Process

1. **Start with seed keywords** — the core topic or product category
2. **Expand with long-tail variations** — more specific phrases with lower competition (e.g., "best running shoes" → "best running shoes for flat feet 2026")
3. **Check search intent** — match content type to what searchers expect:
   - Informational: "how to", "what is", "guide to"
   - Commercial: "best", "top", "review", "vs"
   - Transactional: "buy", "discount", "shop"
4. **Map keywords to funnel stage**:
   - Top of funnel (awareness): broad educational topics
   - Middle of funnel (consideration): comparison and "best of" content
   - Bottom of funnel (decision): product-specific, buying guides

## On-Page SEO Checklist

### Title Tag
- Primary keyword within the first 30 characters if possible
- Under 60 characters total
- Compelling enough to earn clicks in search results

### Meta Description
- Under 155 characters
- Include primary keyword naturally
- Add a call-to-action or value proposition
- Make it specific — avoid generic descriptions

### URL Slug
- Short and descriptive: `/blogs/news/best-running-shoes-flat-feet`
- Use hyphens, not underscores
- Include primary keyword
- Remove stop words (a, the, and, of) unless they improve readability

### Heading Structure
- H1: Blog post title (only one per page — Shopify handles this automatically)
- H2: Major sections (include primary or secondary keywords in at least one)
- H3: Subsections within H2 blocks
- Never skip heading levels (e.g., don't jump from H2 to H4)

### Content Optimization
- First 100 words should include the primary keyword
- Use secondary keywords in H2 headings where natural
- Aim for 1-2% keyword density (don't force it)
- Include LSI (semantically related) terms throughout
- Write alt text for all images with relevant keywords

### Internal Linking
- Link to 2-3 related blog posts within the content
- Link to relevant product pages or collections naturally
- Use descriptive anchor text (not "click here")
- Place important links early in the content

### Image Optimization
- Descriptive file names: `blue-running-shoes-side-view.jpg` not `IMG_4521.jpg`
- Alt text that describes the image and includes keywords where natural
- Compress images for fast loading (Shopify recommends under 200KB)
- Use WebP format when possible

## Content Length Guidelines by Post Type

| Post Type | Ideal Length | Notes |
|-----------|-------------|-------|
| Product spotlight | 400-600 words | Focused, benefit-driven |
| How-to guide | 800-1500 words | Step-by-step, practical |
| Buying guide | 1500-2500 words | Comprehensive, comparison-heavy |
| Listicle | 800-1200 words | Scannable, well-organized |
| Brand story | 600-1000 words | Personal, authentic |
| Seasonal/trend | 600-1000 words | Timely, shareable |

## Shopify-Specific SEO Notes

- Shopify auto-generates the blog URL structure as `/blogs/{blog-handle}/{post-handle}`
- The blog handle is set at the blog level, not per post
- Shopify does not support canonical tag customization natively — use apps if needed
- Use Shopify's built-in image CDN for fast loading
- Blog post excerpts are used for the blog listing page — write them intentionally
- Schema markup (Article structured data) can be added through theme customization

## Measuring Blog Performance

Track these metrics to evaluate blog post success:

- **Organic traffic** — sessions from search to the blog post
- **Keyword rankings** — position for target keywords over time
- **Time on page** — engagement quality indicator
- **Bounce rate** — lower is better for informational content
- **Click-through to products** — how often readers visit product pages
- **Conversion rate** — sales attributed to blog traffic (use UTM parameters)
